# To get started:

1. Clone the app
2. run `yarn` to install dependencies
3. run `yarn dev` to start local server


## RSVP schedule (cron)

RSVP opens automatically at 6:00 AM CT and closes at 10:00 PM CT on match day. See [docs/rsvp-cron.md](docs/rsvp-cron.md) for cron setup (Vercel Cron, CRON_SECRET, or BullMQ/external cron).

## Some of the libraries used:

1. Zustand (State management)
2. Shadcn/ui (headless UI components)
3. Tailwindcss for styling
